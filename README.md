# Rock Paper Scissors

Console-based Rock Paper Scissors game from The Odin Project.

## How to Play

1. Open `index.html` in a browser
2. Open the console (F12)
3. Enter your choice when prompted: rock, paper, or scissor
4. Play 5 rounds against the computer

## What I Learned

- JavaScript functions and scope
- Conditional logic for game rules
- Using `Math.random()` for computer choices
- Returning values from functions instead of using global variables


Part of [The Odin Project](https://www.theodinproject.com/) Foundations course.
